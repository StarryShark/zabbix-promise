Class: Zabbix

Zabbix(opts)

Zabbix class represents the Zabbix API Client to make JSON-RPC 2.0 protocol compliant HTTP POST requests. You instantiate a class with an object that includes the required properties listed below, and then you should call the login method first to get the valid authentication token. Now you are ready to make other Zabbix API calls and don't forget to call the logout method when you are done.
