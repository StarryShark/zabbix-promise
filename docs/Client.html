(async, static) sender(options)

The sender method implements the Zabbix Sender protocol natively in JavaScript. There is no need to authenticate with Zabbix to use sender method. Just pass an object with server, host, key and value properties.
Parameters:
Name Type Description
options object An object contains the payload to send to Zabbix sender.
Properties:
Name Type Attributes Description
port integer <optional>
 Zabbix server port to receive sender traffic. Defaults to 10051.
server string Zabbix server name or IP address
host string Zabbix host as configured in frontend
key string Zabbix host item key
value string | number The value to send
Source:
  • utils.js, line 34
Example
Client.sender({
  server: '127.0.0.1',
  host: 'zabbix.host',
  key: 'item.test.key',
  value: 'Hey there!'
})
